# 🚀 SYNTHETICA — AI World Generation Platform

## 📖 About
Synthetica is an AI-powered platform that generates interactive 3D worlds from text prompts in real-time using WebSocket streaming and 128× compression technology.

## ✨ Features
- **AI-Powered Generation** — Create 3D worlds from text
- **Real-Time Streaming** — WebSocket at 11 FPS
- **128× Compression** — 19.1× faster inference
- **Developer First** — REST + WebSocket APIs

## 🛠️ Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Python, FastAPI, WebSockets
- **Deployment**: Vercel (frontend), Railway (backend)
